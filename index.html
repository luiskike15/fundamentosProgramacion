Fundamentos de programacion

Que es la programacion?

Es dar instrucciones para realizar tareas que resuelvan una necesidad pero eso si.. tienen que ser bien especificos, esto se realiza mediante los lenguajes de programacion, los lenguajes maquina solo acepta 0 y 1.

Que son las variables

Las variables es un valor especifico que nosotros nombramos y esto ocupa un espacio en nuestra memoria ram. y en el lenguaje Js se nombra una variable con el 'Let'.

Por ejemplo:

let edadHijo=20;
let miNombre="peluchin";

Que son los tipos de datos

Es como una clasificacion que le damos a los datos que definimos, estos puedes ser de tipo numerico y dentro de ello pueden ser int que son los numeros enteros o float son los decimales, otro tipo de dato es de texto , que se define como string, otro tipo de dato es booleano , es como un switch es bien un true o false.

Que son los operadores

Son funciones que ralizan operaciones con uno o mas valores. Entre ellos tenemos:

OPERADORES ARTIMETICOS Son las operaciones artiméticas que se utiliza para resolver alguna necesidad. Estos son: La suma(+), resta(-), multiplicacion(*) , división(/) y modulo(%) que es el residuo.

OPERADORES DE ASIGNACION Es utilizado para dar valor a las variables y si es un let se puede reasignar pero una constante no se puede asignar.

OPERADORES DE COMPARACION Se utiliza para comparar a las variables que definimos si es verdad o falso y se utiliza con < > >= <= == .

OPERADORES LOGICOS Es como una toma de desicion en programacion, realiza condiciones a partir de operadores como '&&, que quiere decir que tiene que cumplir tal funcion y tambien tiene que cumplir la otra funcion para que salga true y si no cumple una condicion es false' 'OR, indica que si cumple tan solo una condicion el resto es true '
